Features

Interactive TUI

Keyboard-first terminal UI with table view, flamegraph, and detailed inspection. 60fps rendering.

Flamegraph Visualization

Beautiful flamegraphs showing execution timeline, latency distribution, and span hierarchy.

Cost Tracking

Track token usage and costs across all LLM calls. Understand where your money goes.

HTML Export

Generate self-contained HTML reports for sharing traces with teammates or for documentation.

Language Adapters

Python, Rust, and more. Lightweight adapters that work with any agent framework.

Blazing Fast

Parse 100k spans in under 200ms. Built in Rust for maximum performance.

How It Works

1

Emit Traces

Use our lightweight adapters to emit trace events from your agent. Just a few lines of code.

with span("llm_call", model="claude-3.5"):
  response = call_llm()
2

trace.jsonl

Events are appended to a local trace.jsonl file. Simple, append-only format. No database needed.

{"ts": "...", "name": "llm_call", ...}
{"ts": "...", "name": "tool_call", ...}
3

View & Analyze

Open the file with the TUI or Desktop app. See flamegraphs, timelines, and cost breakdowns.

trace.tui --watch trace.jsonl
# Or use Desktop app
